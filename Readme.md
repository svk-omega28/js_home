### Описание домашнего задания

Реалізуйте функцію removeElement(array, item), щоб видалити елемент item з масиву array.

Наприклад:

Copy code
const array = [1, 2, 3, 4, 5, 6, 7];
removeElement(array, 5 );
console.log(array);
// Результат: [1, 2, 3, 4, 6, 7]
